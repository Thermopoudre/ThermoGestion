2. Données collectées

2.1. Données des utilisateurs atelier

Nous collectons les données suivantes concernant les utilisateurs de l'atelier :

  • Données d'identification : nom, prénom, email, téléphone, rôle dans l'atelier
  • Données de connexion : identifiant, mot de passe (chiffré), tokens d'authentification
  • Données d'activité : connexions, actions effectuées dans le Service, logs d'utilisation
  • Données de facturation : informations de paiement (via Stripe, non stockées sur nos serveurs)
  • Données techniques : adresse IP, type de navigateur, système d'exploitation, résolution d'écran
  • Données de préférences : paramètres utilisateur, préférences d'affichage, langue choisie

2.2. Données de l'atelier

Nous collectons les données suivantes concernant l'atelier :

  • Données d'identification : nom de l'atelier, SIRET (si professionnel), adresse, téléphone, email
  • Données de configuration : paramètres de l'atelier (marges, taux horaire, workflows, etc.)
  • Données de stockage : utilisation du stockage photos (GB utilisés, quotas)

2.3. Données des clients finaux

Les données des clients finaux sont collectées et stockées par l'atelier dans le Service. Ces données comprennent :

  • Données d'identification : nom, prénom, email, téléphone, adresse, type (pro/particulier)
  • Données commerciales : devis, projets, factures, paiements, historique des échanges
  • Données techniques : photos avant/après, documents (devis, factures, rapports)
  • Données de connexion : identifiants portail client (si compte créé), adresse IP

Important : L'atelier est responsable du traitement de ces données conformément au RGPD. L'Éditeur agit en tant que sous-traitant.

2.4. Cookies et technologies similaires

Nous utilisons des cookies et technologies similaires pour :

  • Authentification (session, tokens)
  • Préférences utilisateur (langue, thème)
  • Analytics (Google Analytics, anonymisé RGPD-compliant)

Pour plus d'informations, consultez notre Politique cookies.

3. Finalités du traitement

Les données personnelles sont collectées et traitées pour les finalités suivantes :

  • Exécution du contrat : fourniture du Service, gestion de l'abonnement, facturation, support technique
  • Amélioration du Service : analyse de l'utilisation, correction des erreurs, développement de nouvelles fonctionnalités
  • Communication : envoi d'emails nécessaires au Service (confirmations, notifications, factures), communication marketing (avec consentement)
  • Conformité légale : respect des obligations légales (facturation, conservation données comptables, RGPD)
  • Sécurité : détection des fraudes, prévention des abus, sécurisation du Service
  • Analytics : statistiques d'utilisation (anonymisées), amélioration de l'expérience utilisateur

4. Base légale du traitement

Le traitement de vos données personnelles est fondé sur :

  • Exécution du contrat : traitement nécessaire à l'exécution de l'abonnement et à la fourniture du Service
  • Consentement : traitement basé sur votre consentement explicite (cookies non essentiels, marketing)
  • Obligation légale : traitement nécessaire au respect des obligations légales (facturation, conservation documents comptables)
  • Intérêt légitime : traitement nécessaire aux intérêts légitimes de l'Éditeur (sécurité, amélioration du Service, analytics anonymisées)

5. Durée de conservation

Les données personnelles sont conservées pour les durées suivantes :

  • Données de compte utilisateur : durée de l'abonnement + 3 ans après résiliation (conformité légale)
  • Données de facturation : 10 ans (obligation légale fiscale française)
  • Données clients finaux (projets, devis, factures) : durée de l'abonnement + 10 ans après résiliation (obligation légale comptable)
  • Logs d'activité : 90 jours (debug, sécurité)
  • Logs d'audit : 2 ans (traçabilité actions critiques)
  • Données marketing : jusqu'à retrait du consentement ou 3 ans d'inactivité
  • Données supprimées : suppression définitive après durée de conservation + période de grâce de 30 jours (export possible)

À l'expiration de ces durées, les données sont supprimées définitivement ou anonymisées.

6. Destinataires des données

6.1. Destinataires internes

Les données personnelles sont accessibles uniquement aux personnes autorisées de l'Éditeur dans le cadre de l'exécution du Service (support technique, administration).

6.3. Partages avec tiers

Les données personnelles ne sont pas vendues, louées ou partagées avec des tiers à des fins commerciales.

Les données peuvent être partagées uniquement dans les cas suivants :

  • Obligation légale (réquisition judiciaire, contrôle fiscal)
  • Protection des droits (en cas de violation des CGU/CGV, utilisation frauduleuse)
  • Avec votre consentement explicite

8. Vos droits

Conformément au RGPD et à la loi Informatique et Libertés, vous disposez des droits suivants :

8.1. Droit d'accès

Vous avez le droit d'obtenir la confirmation que vos données personnelles sont traitées et d'accéder à ces données. Vous pouvez accéder à vos données depuis votre dashboard ou demander un export complet.

8.2. Droit de rectification

Vous avez le droit de rectifier vos données personnelles inexactes ou incomplètes. Vous pouvez modifier vos données depuis votre dashboard ou en contactant le support.

8.3. Droit à l'effacement (droit à l'oubli)

Vous avez le droit d'obtenir l'effacement de vos données personnelles dans les cas suivants :

  • Les données ne sont plus nécessaires aux finalités pour lesquelles elles ont été collectées
  • Vous retirez votre consentement (si traitement basé sur consentement)
  • Vous vous opposez au traitement (si traitement basé sur intérêt légitime)
  • Les données ont été traitées illicitement

Exception : L'effacement peut être limité par des obligations légales de conservation (factures : 10 ans, etc.).

8.4. Droit à la limitation du traitement

Vous avez le droit de limiter le traitement de vos données personnelles dans certains cas (contestation de l'exactitude, opposition au traitement, etc.).

8.5. Droit à la portabilité

Vous avez le droit de recevoir vos données personnelles dans un format structuré, couramment utilisé et lisible par machine, et de les transmettre à un autre responsable du traitement. Vous pouvez exporter vos données depuis votre dashboard (format CSV/JSON).

8.6. Droit d'opposition

Vous avez le droit de vous opposer au traitement de vos données personnelles pour des motifs légitimes (marketing, profiling). Vous pouvez retirer votre consentement depuis votre dashboard ou en contactant le support.

8.7. Droit de retrait du consentement

Vous avez le droit de retirer votre consentement à tout moment pour les traitements basés sur le consentement (cookies non essentiels, marketing). Le retrait du consentement n'affecte pas la licéité du traitement effectué avant le retrait.

8.8. Droit de définir des directives post-mortem

Vous avez le droit de définir des directives concernant le sort de vos données personnelles après votre décès (conservation, effacement, communication à un tiers désigné).

11. Sécurité des données

L'Éditeur met en œuvre des mesures techniques et organisationnelles appropriées pour assurer la sécurité des données personnelles :

  • Chiffrement : données en transit (HTTPS/TLS), données sensibles en base (chiffrement AES)
  • Authentification forte : 2FA obligatoire pour rôles Owner/Admin
  • Isolation des données : multi-tenancy strict avec Row Level Security (RLS Supabase)
  • Sauvegardes : sauvegardes quotidiennes automatiques, retention 30-90 jours
  • Monitoring : détection d'intrusion, logs de sécurité, alertes automatiques
  • Accès restreint : accès aux données limité aux personnes autorisées (principe du moindre privilège)
  • Tests sécurité : scans vulnérabilités automatiques, tests d'intrusion réguliers

Malgré ces mesures, aucune méthode de transmission ou de stockage n'est totalement sécurisée. L'Éditeur ne peut garantir une sécurité absolue des données.
